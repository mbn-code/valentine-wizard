## How it works
1. **Wizard:** Use the `/wizard` interface to select a plan and personalize the content.
2. **URL-based Storage:** All configurations are compressed and stored directly in the URL hash.
3. **No Database:** Privacy-focused and cost-effective. No user data is stored on a server.

## Deployment
Deploy to Vercel/Netlify as a static site. The URL-based configuration makes it horizontally scalable and zero-maintenance.
